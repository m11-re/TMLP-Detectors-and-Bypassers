#### Riru

Riru is a rooting-layer system module that offers rich application programming interfaces (APIs) to other system modules or plugins. 

The original Xposed, EdXposed, and early versions of LSPosed modules require riru-core to work normally. 

The link to the official GitHub repository of Riru is [https://github.com/RikkaApps/Riru](https://github.com/RikkaApps/Riru). 

Currently, this repository is publicly archived. Most users have migrated to [Zygisk](../Zygisk) for more precise control, better performance, and greater concealment, as suggested.

#### Riru

Riru 是一个 root 层的系统模块，它为其他系统模块或插件提供丰富的应用程序编程接口。

最初的 Xposed、EdXposed 和早期版本的 LSPosed 模块需要 riru-core 才能正常工作。

Riru 的官方 GitHub 存储库链接为 [https://github.com/RikkaApps/Riru](https://github.com/RikkaApps/Riru)。

目前，该存储库已被设为公开且归档状态；大多数用户已按照建议迁移到控制更精确、性能更好和隐蔽性更强的 [Zygisk](../Zygisk)。
